# svm3udn

Программа умеет скачивать и сохранять HLS (m3u m3u8 файл плейлиста) потоки. Как записи, так и трансляции.
Пока не работает с мастер плейлистами (это плейлисты, содержащие другие плейлисты). 
Работает только с прямыми, содержащими ссылки на непосредственно потоки данных.

Формат вызова (пример):
svm3udn -proxy="" -rec=1 -dl=1 -frseg=0 -fromsegfile=0 -savesegfile=0 -url=https://ott-live.olympicchannel.com/out/u/OC13_1.m3u8?m=1561977273

-proxy= позволяет указать прокси в формате, например: "socks4://xxx.xxx.xxx.xxx"
-rec= 1-плейлист записи, 0-прямая трансляция
-dl= 1-необходимо сохранять на диске (файл test.ts)
-frseg= может указывать номер начального сегмента с которого начинать сохранять
-fromsegfile= 1-указывает на то, что будет использоваться файл сегментов, который может быть записан командой -savesegfile
-savesegfile= 1-запись файла сегментов (файл содержит абсолютные ссылки на потоки данных)
-url= url плейлиста
